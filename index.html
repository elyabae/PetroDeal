О проекте

Название проекта: Прогноз сезонных затрат на транспортировку нефтепродуктов

Цель: Разработка веб-приложения, которое помогает анализировать и прогнозировать расходы на перевозку нефтепродуктов в зависимости от сезона и других факторов. Сайт ориентирован на использование внутри вымышленного нефтетранспортного предприятия.

Функционал сайта

  • Форма обратной связи — пользователь может оставить сообщение, отзыв или задать вопрос.
  • Аналитическая страница — отображение отправленных сообщений в виде таблицы (динамически с помощью PHP и MySQL).
  • Навигация — простая и понятная, с переходами между главной, аналитикой и контактами.

Технологии

В проекте использованы:

  • HTML + CSS — для вёрстки и стилей
  • PHP — для серверной логики
  • MySQL — для хранения данных формы

Почему это важно?

Такой сайт — это первый шаг к цифровизации процессов на предприятии. Он может быть расширен: подключение графиков, расширенной статистики и прогнозов на основе реальных данных.
